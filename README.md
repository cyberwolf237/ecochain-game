**An interactive educational game about Science & Nature for Teens & Adults**

Build and manage ecosystems, learn real science, and save the planet! 🌱

## 📱 Platform Support
- ✅ Android Mobile
- ✅ Windows/Mac/Linux PC

---

## 🎮 Game Overview

**EcoChain** is an engaging puzzle-adventure game where players:
- Build balanced ecosystems by placing plants, herbivores, predators, and decomposers
- Learn real science concepts about food chains, biodiversity, and conservation
- Face environmental challenges like droughts, extreme weather, and human impact
- Unlock new biomes (rainforest, savanna, arctic, ocean, desert)
- Progress through 50+ scientifically-accurate levels

## 🚀 Getting Started

### Prerequisites
1. **Godot 4.x** - Download from https://godotengine.org
2. **Git** - For version control
3. **Text Editor/IDE** - VS Code recommended

### Installation

1. Clone the repository:
```bash
git clone https://github.com/cyberwolf237/ecochain-game.git
cd ecochain-game
```

2. Open the project in Godot:
   - Launch Godot
   - Select "Import"
   - Choose the `game/` folder
   - Click "Import & Edit"

3. Run the game:
   - Press F5 or click the Play button
